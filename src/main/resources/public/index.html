JSON Web Key Generator

mkjwk

A JSON Web Key (JWK) is a cryptographic key or keypair expressed in JSON format. This site offers a mechanism to easily generate random keys for use in servers and other projects.

This server will never log or store any generated keys. The source code for this server is available on GitHub for inspection and re-use. If you would like to generate your own key locally (so as not to trust), you can use the commandline utility version of the library that powers this site.

  • RSA
  • Elliptic Curve
  • Shared Secret

Keypair

Contains both public and private keys.

Keypair set

Wraps keys in a key set.

Public key

Contains only the public key.
